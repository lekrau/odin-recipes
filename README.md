# odin-recipes
https://www.theodinproject.com/lessons/foundations-recipes

This project is a simple website with three of the recipes I eat on a daily basis.
Currently it is made of HTML only, using elements like headings, images, anchors (links), paragraphs, unordered lists, and ordered lists. Besides using these elements I practiced my ability to organize a project directory, creating a github repository and making meaningful commit messages.
